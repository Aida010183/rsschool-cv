что такое панель руководителя

Информационная витрина, функция которой заключается в том, чтобы интегрировать информацию из различных информационных корпоративных систем компании и отражать её в ракурсе проектного управления.

Интеграция SmartPlanner с Microsoft Project

Из Microsoft Project данные передаются в «Панель Руководителя», что дает возможность оперативно получить данные для управленческой отчётности, с которыми работают управленцы. На основании полученной отчетности менеджеры компании, имея полные данные, могут чётко и быстро принимать решения. ​
